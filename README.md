## 🌟 Features

### For Travelers

-   Browse authentic experiences\
-   Advanced search & filtering\
-   Secure bookings with availability logic\
-   Booking management dashboard\
-   Reviews & ratings\
-   Personalized dashboard

### For Guides

-   Create & manage experiences\
-   Flexible availability rules\
-   Booking request management\
-   Profile verification system\
-   Earnings tracking\
-   Public profile showcase

### General Features

-   Role-based authentication\
-   Responsive UI (Tailwind CSS)\
-   Email notifications\
-   Django Admin panel\
-   Multilingual booking support\
-   Image uploads with Pillow\
-   Moderated review system

## 🛠 Tech Stack

-   **Backend:** Django 6.0.1\
-   **Frontend:** HTML5, JavaScript, Tailwind CSS 4.1.18\
-   **Database:** SQLite (dev) / PostgreSQL (recommended for
    production)\
-   **Image Processing:** Pillow 12.1.0\
-   **Environment Management:** django-environ\
-   **Build Tools:** Node.js, PostCSS, Autoprefixer\
-   **Deployment Ready:** Heroku / AWS / DigitalOcean

## 📦 Installation

### Prerequisites

-   Python 3.8+\
-   Node.js 14+\
-   Git

### Setup

``` bash
git clone https://github.com/AmeriK88/exploideo.git
cd exploideo

python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

pip install -r requirements.txt
npm install

python manage.py migrate
python manage.py createsuperuser

npm run tw:build
python manage.py runserver
```

Visit: http://127.0.0.1:8000
